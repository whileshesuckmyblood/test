PET-Проект, Джуниор ДевОпс

Идея проекта - разработка и развертывание ТГ-бота с готовой прод. инфраструктурой.
Цель - продемонстрировать навыки работы со стеком.

Стек:
- python 3.11 + aiogram 3.13.0 - логика бота, работает через вебхуки.
- nginx (reverse proxy + lets encrypt SSL) - обрабатывает HTTPS запросы и перенаправляет их в контейнер бота.
- Docker - контейнеризация приложения, гарантирует одинаковую стабильную работу на разных устройствах.
- docker compose - оркестратор контейнеров, в проекте используется два контейнера nginx и tg-bot, управляет запуском, сетями и зависимостями.
- GitHub Actions (CI/CD) - используется автоматическая сборка, тестирование и деплой приложения на VPS через SSH, при каждом коммите нашего кода в репозиторий.
- VPS (Linux) - Ubuntu24.04.

Как работает: 
Пользователь пишет боту(в моем случае бот - просто дублирует сообщения пользователя), API телеграма направляет сообщение HTTPS запрос на мой домен, nginx принимает запрос на 443 порту, производит расшифровку через SSL(lets encrypt), и после этого перенаправляет запрос по внутренней сети Докера к контейнеру с ботом на 8080 порту. Бот обрабатывает данные и возвращает ответ обратно через прокси.
telegram -> HTTPS(443) -> nginx (reverse proxy + SSL) --(внутрення сеть докера)--> контейнер бота (порт 8080) -> ответ пользователю

CI/CD
CI(непрерывная интеграция) - при пуше кода в ветку main, запускается workflow, Actions проверяет репозиторий, устанавливает python 3.11 и зависимости, запускает тесты через pytest(в моем случае тест - просто print("hello!")), собирается образ бота с тегом коммита SHA, образ пушится в реестр контейнеров гитхаба(GHCR) для последующего деплоя.
CD(непрерывное развертывание) - после того, как успешно прошел этап интеграции - начинается этот, деплой кода на VPS через SSH, происходит логин в GHCR и пулл последнего docker-образа. Затем останавливается текущий контейнер бота, и перезапускается свежий.

Благодаря этим двум этапам, каждое изменение кода автоматически обновляет бота на сервере без ручного вмешательства, что очень удобно!
